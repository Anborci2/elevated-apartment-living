## Component Contracts

### Hero Section (`components/sections/Hero.tsx`)

**Structure:**
- Outermost: `<section>` with `relative min-h-screen flex items-center justify-center`
- Background: `<div>` with `absolute inset-0` containing `next/image` (fill, object-cover)
- Overlay: `<div>` with `absolute inset-0 bg-luxury-black/60`
- Content: `<div>` with `relative z-10 text-center px-6 max-w-4xl mx-auto`

**Background Image:**
- Source: Unsplash — search `luxury apartment interior dark moody` or `modern apartment living room evening`
- Preferred tones: dark walls, warm lamp light — blends with 60% black overlay
- Suggested URL: `https://images.unsplash.com/photo-1545324418-cc1a3fa10c00?w=1920&q=80` (fallback — implementer may substitute better match)
- `next/image` props: `fill`, `priority`, `quality={90}`, `className="object-cover object-center"`

**Headline:**
- `<h1>` with `font-playfair text-display-mobile lg:text-display font-bold text-luxury-text-primary tracking-tight leading-tight`
- Copy: "Find Your Perfect Denver Home"

**Sub-headline:**
- `<p>` with `mt-4 font-inter text-base lg:text-lg text-luxury-text-secondary max-w-2xl mx-auto leading-relaxed`
- Copy: "Denver's apartment locating expert — tell us what you need, and we'll find it."

**CTA Button:**
- `<a href="#lead-form">` with `inline-block mt-8` wrapper
- Button: same spec as Phase 1 CTA — `border border-luxury-gold text-luxury-gold bg-transparent px-8 py-3 text-sm font-semibold tracking-widest uppercase rounded-none transition-all duration-200 hover:bg-luxury-gold hover:text-luxury-black`
- Smooth scroll: `scroll-behavior: smooth` on `<html>` (or `onClick` handler with `scrollIntoView`)

**Header overlay:** No `pt` on Hero. Header is fixed z-50; hero `min-h-screen` fills the viewport below it naturally.

## Accessibility

Inherited rules from Phase 1, plus Phase 2 additions:

- All contrast ratios from Phase 1 confirmed — no new color combinations introduced
- `#F5F0E8` on `#0A0B0D` overlay (hero): ratio ~14:1 ✓ (text on 60% darkened image)
- `#C9A96E` (step numbers, footer link) on `#0A0B0D`: ratio ~5.8:1 ✓
- `#A8A29E` on `#0D1B2A`: ratio ~5.1:1 ✓
- All `next/image` elements require descriptive `alt` text
- Hero `<section>` gets `aria-label="Hero"` (or equivalent landmark)
- How It Works step numbers are decorative — screen readers read step title as the primary h3
- Footer `<a href="mailto:...">` has readable link text (full email address)
- CTA scroll anchor: `href="#lead-form"` with smooth scroll behavior
